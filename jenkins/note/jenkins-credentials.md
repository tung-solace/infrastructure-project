### Credentials in Jenkins
Credentials is a plugin that allows Jenkins to manage and centralize the credentials management.

### Credential Scope
There are two different Jenkins Credentials scope: 
1. System Credentials: 
- System Credentials only avaialble on the Jenkins server. Not for Jenkins jobs

2. Global Credentials: 
- Every where accessible. 

### Credential types: 
There are different credential types: 
1. Username & Password
2. Certificate
3. Secret File
And other types that are available...
